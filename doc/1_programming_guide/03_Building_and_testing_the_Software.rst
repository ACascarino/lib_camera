Requirements
------------------------------------------

Hardware requirements:
^^^^^^^^^^^^^^^^^^^^^^^
- XCORE.AI EVALUATION KIT (XK-EVK-XU316)
- Camera module
- Camera ribbon connector
- 2x Micro USB cable (Power supply and xTag)
- xTag debugger and cable

Software requirements:
^^^^^^^^^^^^^^^^^^^^^^^
- XMOS tools (15.2.1): `SW_TOOLS`_
- CMake, Ninja (Windows)
- Python 3.7 or later
